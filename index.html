파일이 브라우저를 떠나지 않는 이유

바로PDF에는 파일을 받는 서버 자체가 없습니다. 페이지를 여는 순간 PDF 처리 엔진(오픈소스 pdf-lib, Mozilla의 PDF.js)이 여러분의 브라우저에 내려와, 모든 작업이 내 컴퓨터의 메모리 안에서만 실행됩니다.

  • 업로드 없음 — 개발자 도구(F12) 네트워크 탭을 열어 확인해 보세요. 파일 전송 요청이 없습니다.
  • 무제한 무료 — 서버 비용이 들지 않으므로 용량 제한, 하루 횟수 제한, 워터마크가 전혀 없습니다.
  • 원본 보존 — 처리 결과는 항상 새 파일로 다운로드되고 원본은 건드리지 않습니다.
  • 오프라인 동작 — 페이지 로딩 후에는 인터넷이 끊겨도 계속 사용할 수 있습니다.

자주 묻는 질문

정말 파일이 서버로 올라가지 않나요?

네. 처리 엔진이 브라우저에서 직접 실행되는 구조라 파일을 받을 서버가 존재하지 않습니다. 브라우저 개발자 도구의 네트워크 탭에서 직접 확인할 수 있습니다.

회원가입이나 설치가 필요한가요?

아니요. 프로그램 설치, 회원가입, 이메일 입력 없이 페이지를 열자마자 바로 사용할 수 있습니다.

파일 크기나 사용 횟수 제한이 있나요?

없습니다. 처리 속도는 사용 중인 기기 성능을 따르며, 수백 페이지 PDF도 처리할 수 있습니다.

어떤 브라우저에서 쓸 수 있나요?

최신 Chrome, Edge, Safari, Firefox, 삼성 인터넷 등 대부분의 최신 브라우저를 지원합니다. 스마트폰에서도 동작합니다.
