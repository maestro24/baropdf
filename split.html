PDF 분할하는 방법 (3단계)

  1. 분할할 PDF를 위 상자에 끌어다 놓으면 총 페이지 수가 표시됩니다.
  2. "범위 추출"에서 1-3,5,7- 처럼 입력하거나, "개별 분할"을 선택합니다.
  3. 버튼을 누르면 추출 PDF 또는 낱장 PDF ZIP이 바로 다운로드됩니다.

자주 묻는 질문

페이지 범위는 어떻게 입력하나요?

쉼표로 구분해 입력합니다. 예: 1-3,5,7- 은 1~3페이지, 5페이지, 그리고 7페이지부터 끝까지를 뜻합니다. "-3"은 처음부터 3페이지까지입니다.

분할하면 파일이 여러 개 생기는데 어떻게 받나요?

개별 분할을 선택하면 페이지마다 1개씩 만들어진 PDF들이 ZIP 파일 하나로 묶여 다운로드됩니다.

분할할 때 파일이 서버로 업로드되나요?

아니요. 분할과 추출 모두 브라우저 안에서만 처리되며 파일은 어디로도 전송되지 않습니다.

같은 페이지를 여러 번 넣을 수 있나요?

네. 예를 들어 1,1,2 로 입력하면 1페이지가 두 번 들어간 PDF가 만들어집니다.

파일이 안전한 이유

분할 엔진(오픈소스 pdf-lib)이 브라우저 안에서 직접 실행됩니다. 파일을 받는 서버가 없으므로 업로드 자체가 불가능한 구조이며, 원본 파일은 절대 변경되지 않습니다.
