    페이지 정리하는 방법 (3단계)

    1. 정리할 PDF를 위 상자에 끌어다 놓으면 페이지 썸네일이 표시됩니다.
    2. 썸네일을 드래그해 순서를 바꾸고, ⟲⟳ 버튼으로 회전, ✕ 버튼으로 삭제합니다.
    3. "변경 사항 저장"을 누르면 정리된 새 PDF가 다운로드됩니다.

    자주 묻는 질문

    페이지 순서는 어떻게 바꾸나요?

    PDF를 올리면 페이지 썸네일이 격자로 표시됩니다. 썸네일을 마우스로 끌어 원하는 위치에 놓으면 순서가 바뀝니다.

    페이지 회전과 삭제도 되나요?

    네. 각 썸네일 아래 버튼으로 왼쪽/오른쪽 90도 회전과 삭제가 가능하며, 저장 버튼을 누르면 변경 사항이 적용된 새 PDF가 다운로드됩니다.

    원본 파일이 변경되나요?

    아니요. 결과는 항상 새 파일로 다운로드되며 원본 PDF는 그대로 보존됩니다. 처음부터 다시 편집하려면 되돌리기 버튼을 누르세요.

    편집할 때 파일이 서버로 업로드되나요?

    아니요. 썸네일 생성과 저장이 전부 브라우저 안에서 실행되며 파일은 어디로도 전송되지 않습니다.

    PDF에서 특정 페이지만 삭제하려면 어떻게 하나요?

    지우려는 페이지 썸네일의 ✕ 버튼을 누른 뒤 저장하면 그 페이지가 빠진 새 PDF가 만들어집니다. 필요 없는 표지나 빈 페이지를 빼고 정리할 때 편리합니다.

    뒤집혀 있는 PDF 페이지를 회전해서 저장할 수 있나요?

    네. 각 페이지를 왼쪽·오른쪽 90도로 돌린 뒤 저장하면 방향이 바로잡힌 PDF로 내려받아집니다. 가로로 스캔되거나 뒤집힌 문서를 똑바로 맞출 때 사용하세요.

    PDF 페이지 정리가 필요한 순간

    필요 없는 표지나 빈 페이지를 삭제할 때, 거꾸로 스캔된 문서를 회전해 바로 세울 때, 뒤섞인 페이지의 순서를 바꿔 다시 정리할 때 썸네일을 보며 한눈에 편집할 수 있습니다. 여러 페이지를 지우고 돌리고 옮긴 뒤 한 번에 저장하면 변경 사항이 반영된 새 PDF가 만들어지고, 원본은 그대로 남습니다. 모든 PDF 페이지 편집이 브라우저 안에서 이루어져 문서가 외부로 전송되지 않습니다.

    파일이 안전한 이유

    썸네일은 Mozilla PDF.js가, 저장은 pdf-lib이 브라우저 안에서 직접 처리합니다. 파일을 받는 서버가 없어 업로드 자체가 불가능한 구조이며, 원본 파일은 절대 변경되지 않습니다.
